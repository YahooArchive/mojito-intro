Mojito Request Flow

  1. Middleware (Express)
  2. Routing (routes.json)
  3. App Configuration (application.json)
  4. Mojits (server then client)
    • Controllers (controller.{affinity}.js)
    • Addons and Autoloads ({name}.{affinity}.js)
    • Mojit Configuration (defaults.json and definition.json)
    • Models ({name}.{affinity}.js)
    • Views ({action}.{engine}.html)
    • Binders ({action}.js)
    • Assets (CSS, images, and non-YUI libs)
