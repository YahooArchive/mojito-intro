Middleware

  • Middleware is built on Express and only executes on the server.
  • Middleware has limited access to the rest of Mojito.
  • Mojito ships with several pieces of middleware including contextualizer, static asset handler, request tunnel, body parser, cookie parser, and router.
  • A combination of built-in and custom middleware can be configured in application.json.
  • This is the only place where you'll recognize that node.js is being used.

module.exports = function (req, res, next) {
    // do something
    next();
};
    

"middleware": [
    "./middleware/src-handler.js"
]
