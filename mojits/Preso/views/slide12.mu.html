Binders

  • Binders are the sole access point to controllers and the rest of Mojito in the client.
  • Binders have access to autoloads and MojitProxy, which wraps the controller and configuration.
  • Binders provide a messaging interface for cross-mojit communication via subscribe and listen.

YUI.add('FooBinder', function(Y, NAME)
{
    Y.namespace('mojito.binders')[NAME] =
    {
        init: function(mojitProxy) {
            this.proxy = mojitProxy;
        },
        
        bind: function(node) {
            // set up some listeners
            // node is the Y.Node containing id="{{mojit_view_id}}" in the view
        },
        
        onRefreshView: function(node) {
            // do something after refreshView
        }
    };
}, '0.0.1', {requires: ['mojito-client']});
