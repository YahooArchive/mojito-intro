Mojito Terminology

action context
The central access point to the rest of Mojito in a controller.
affinity
Determines where a resource will be executed; client, server, or common.
binder
Binders only execute in the client and provide access to the controller.
context
A set of key/value pairs which is used to resolve configuration.
dimensions
A set of keys and their possible values which are used to set context.
view
Template used to render the final output.
