Assets

  • Assets are CSS files, images, or non-YUI libs. They have no affinity.
  • Mojito is not aware of these types of files and will not deploy them automatically.
  • Assets may be included in application.json or a controller. In either case the HTMLFrameMojit must be used.

"specs": {
    "foo": {
        "type": "HTMLFrameMojit",
        "config": {
            ...
            "assets": {
                "top": {
                    "css": [
                        "/static/assets/app.css"
                    ...
    

ac.assets.addCss('/static/Foo/assets/foo.css', 'top');
