Installing Mojito

First Install node.js, then:


$ sudo npm i mojito -g
$ sudo npm i supervisor -g

Git These Slides

Browse at GitHub or Install git, then:


$ git clone git://github.com/gzip/mojito-intro.git && cd mojito-intro
$ npm i highlight
$ supervisor -w . -e 'js|json|html' start.js

And open your browser to http://localhost:8777/
