What is Mojito?

  • Mojito is an open source MVC framework that runs on the client and/or server via node.js.
  • The core component in Mojito is a mojit (not quite a module, not quite a widget).

Mojito Features

  • Mojito is built on YUI 3 (and expects components to be built on YUI 3).
  • Mojito can execute components on the client, server, or both.
  • Mojito supports a rich configuration framework.
  • Mojito's default view engine is Mustache.
  • Mojito supports inter-mojit communication via broadcasting.
  • Mojito does routing, localization and lazy loading of mojits.
  • Mojito comes with a unit testing framework (YUI Test).
  • Mojito is open source!
