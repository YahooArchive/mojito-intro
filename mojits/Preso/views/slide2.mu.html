Mojito Features

  • Mojito is built on YUI 3 (and expects components to be built on YUI 3).
  • Mojito can execute components on the client, server, or both.
  • Mojito supports a rich configuration framework.
  • Mojito's default view engine is Mustache.
  • Mojito does routing.
  • Mojito does localization.
  • Mojito supports lazy loading of mojits.
  • Mojito comes with a unit testing framework (YUI Test).
